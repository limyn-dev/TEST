Clé de produit Windows
Votre clé de produit Microsoft Windows est le code de 25 chiffres imprimé sur la couverture de votre manuel Pour bien commencer avec Windows 98. La clé de produit est imprimée au-dessus du code-barres, comme dans l'exemple ci-dessous.

Entrez la clé de produit telle qu'elle apparaît sur la couverture de votre manuel Pour bien commencer avec Windows 98.

- - - -
Si vous n'avez pas de clé de produit, cliquez sur Ignorer.
Pour continuer entrez la clé de 25 chiffres et cliquez sur SUIVANT.
